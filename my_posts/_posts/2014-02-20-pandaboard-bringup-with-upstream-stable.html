Pandaboard bringup with upstream stable u-boot and linux kernel



1) Prepare SD card for pandaboard:
Insert card into card reader connected to linux host machine (unmount card lets say its /dev/sdc*) and run
$ sudo pandaboard_make_card.sh /dev/sdc
It will create two partitions. Eject card and re-insert into card reader
both partitions will be mounted as
/media/boot
/media/MEL

2)  Copy Rootfs to SD card:
Copy MEL rootfs to /media/MEL
$ sudo tar -xjf /home/abbas/core-image-atp-mx6q.tar.bz2 -C /media/MEL/

3) Building upstream u-boot:
$ git remote add upstream git://git.denx.de/u-boot.git
$ git fetch upstream
$ git checkout upstream/master
$ make CROSS_COMPILE=arm-none-linux-gnueabi- distclean
$ make CROSS_COMPILE=arm-none-linux-gnueabi- omap4_panda_config
$ make CROSS_COMPILE=arm-none-linux-gnueabi-
$ sudo cp MLO u-boot.img /media/boot/

4) Building upstream kernel:
$ git remote add linux-stable git://git.kernel.org/pub/scm/linux/kernel/git/stable/linux-stable.git
$ git fetch linux-stable
$ git checkout linux-stable/linux-3.13.y
$ make CROSS_COMPILE=arm-none-linux-gnueabi- ARCH=arm -j4 omap2plus_defconfig
Enable any required configuration:
$ make CROSS_COMPILE=arm-none-linux-gnueabi- ARCH=arm -j4 menuconfig
$ make CROSS_COMPILE=arm-none-linux-gnueabi- ARCH=arm -j4
$ sudo cp arch/arm/boot/zImage arch/arm/boot/dts/omap4-panda.dtb /media/boot/
$ sudo rm -rf /media/MEL/lib/modules/*
$ sudo make ARCH=arm CROSS_COMPILE=arm-none-linux-gnueabi- modules_install INSTALL_MOD_PATH=/media/MEL/

5) Boot pandaboard:

i) Eject SD card from host and insert into pandaboard
ii) power-up panda board, stop at autoboot

abbas@abbas:panda$ cat pandaboard_make_card.sh
#! /bin/sh

export LC_ALL=C

if [ $# -ne 1 ]; then
echo "Usage: $0 "
exit 1;
fi

DRIVE=$1

dd if=/dev/zero of=$DRIVE bs=1024 count=1024

SIZE=`fdisk -l $DRIVE | grep Disk | grep bytes | awk '{print $5}'`

echo DISK SIZE - $SIZE bytes

CYLINDERS=`echo $SIZE/255/63/512 | bc`

echo CYLINDERS - $CYLINDERS

{
echo ,9,0x0C,*
echo ,,,-
} | sfdisk -D -H 255 -S 63 -C $CYLINDERS $DRIVE

sleep 1

if [ -b ${DRIVE}1 ]; then
umount ${DRIVE}1
mkfs.vfat -F 32 -n "boot" ${DRIVE}1
else
if [ -b ${DRIVE}p1 ]; then
umount ${DRIVE}p1
mkfs.vfat -F 32 -n "boot" ${DRIVE}p1
else
echo "Cant find boot partition in /dev"
fi
fi

if [ -b ${DRIVE}2 ]; then
umount ${DRIVE}2
mke2fs -j -L "MEL" ${DRIVE}2
else
if [ -b ${DRIVE}p2 ]; then
umount ${DRIVE}p2
mke2fs -j -L "MEL" ${DRIVE}p2
else
echo "Cant find rootfs partition in /dev"
fi
fi
